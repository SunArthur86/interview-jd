# 📝 JD 面试题库

针对一线大厂 JD（岗位描述）深度拆解的面试题库，**401 道题**，覆盖蚂蚁风控、拼多多多业务线、Java 后端架构师等方向，每题含费曼快记、第一性原理、苏格拉底式追问。

## 📚 题目分类（共 401 道）

| 分类 | 题数 | 内容 |
|------|------|------|
| 🐜 **蚂蚁风控** | 40 | JVM/GC、并发、MySQL、Redis、分布式事务、风控业务、AI Agent |
| 🍊 **拼多多 AI 中台** | 40 | 线程池、Spring Boot、Kafka、模型服务、推理优化、AI Harness |
| 📦 **拼多多内容** | 40 | 内容审核、推荐、搜索、Feed 流、RAG |
| 🚚 **拼多多供应链** | 40 | 库存扣减、并发锁、订单状态机、MySQL、分布式事务 |
| 🛒 **拼多多交易** | 40 | 秒杀、支付回调、优惠券、账户对账、热点保护 |
| ☕ **Java 后端架构师** | 201 | JVM/并发/Spring/微服务/MySQL/Redis/Kafka/架构设计/AI Agent |

## ✨ 特色

- **费曼快记**：每题用大白话讲清楚本质
- **第一性原理**：每题触及最根本的公理
- **层层递进**：从表面 → 机制 → 算法 → 应用 → 本质
- **贴合实战**：所有题目结合蚂蚁风控真实场景
- **AI 转型**：覆盖 AI Agent / Harness / Infra / FDE 全链路
